# Plan de Implementación Estratégica AXIS PRO - Evolución TRL

Este documento detalla la hoja de ruta estratégica para la consolidación de AXIS PRO como una plataforma industrial de vanguardia.

---

## 1. Módulo 1: Supply & Quality (Consolidación)
- [ ] **Finalización de Trilla:** Implementar el cálculo automático del Factor de Rendimiento en el backend (Server Actions) para evitar manipulaciones de datos.
- [ ] **Protocolo SCA:** Garantizar la vinculación permanente y segura de los datos de catación al ID del lote.
- **Estado Actual:** TRL 6 -> **Objetivo:** TRL 7 (Listo para demostración operativa).

## 2. Módulo 2: Roast Intelligence (Prioridad Inversionista)
- [x] **Capa de Comparación (Ghost Profile):** Funcionalidad para superponer curvas maestras sobre tuestes en vivo.
- [x] **Asistente de Perillas:** Lógica de recomendación basada en desviaciones detectadas (Server-side logic).
- **Estado Actual:** TRL 7 (Consolidado e IP Protegida).

## 3. Módulo 3: Global Trade
- [x] **Generador de Manifiestos:** Exportación en PDF del "Pasaporte del Café" con datos integrados (Module 1, 2 & Degassing).
- [x] **QR de Trazabilidad:** Interfaz de consulta internacional vinculada al lote físico.
- [x] **Archivo Histórico:** Visualización de reportes guardados en la nube.
- **Estado Actual:** TRL 7 (Consolidado y Demostrable).

## 4. Módulo 4: Retail Connect (Cierre del Círculo)
- [x] **Gestión de Inventario Terminado:** Conversión de tueste a unidades de retail (250g/500g).
- [x] **Visualización para Consumidor:** Interfaz móvil para storytelling y perfil de sabor vía QR (B2C).
- **Estado Actual:** TRL 7 (Operativo y Demostrable).

---

## 5. Master Control: Demo Unlock In-Situ
- [x] **Activación Dinámica:** Capacidad de desbloquear módulos bajo llave "on-the-fly" para presentaciones comerciales.
- [x] **Simulación de Up-selling:** Transición visual de estado 'Locked' a 'Active' para demostrar el valor de capas premium.
